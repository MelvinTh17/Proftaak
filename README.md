
# Prometheus x Zammad

Haal api gegevens van prometheus en maak automatisch support tickets aan

## Waarom?

Omdat het integreren van prometheus met elastic stack moeizaam gaat is het makkelijker om een script te gebruiken die automatisch gegevens uitleest en doorstuurt.


## Features

- Leest CPU en RAM uit van de prometheus API
- Maakt automatische een ticket bij het overschrijden van een waarde
- Print overzichtelijk in een tabel


## Roadmap
